# MULTI-BRAND-MARKETING-CAMPAIGN-PERFORMANCE-DASHBOARD
A Multi-Brand Marketing Campaign Performance Dashboard is a centralized visual tool that tracks and compares the effectiveness of marketing campaigns across different brands.
# Features
Multi- brand campaign analysis
Performance comparison
Interactive dashboard
Data visualization

# Tools Uused
 Power BI
 Excel
 SQL

# Insights
Best performing campaigns
Low performing campaigns
Helps in better decision making
